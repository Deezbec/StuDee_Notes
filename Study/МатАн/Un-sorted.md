## Практика кванторов
`$AA x,y in RR => [x=y || x<y || x>y]$`
## Воспоминания о модуле
```asciimath
abs(x) = {}

-x,x < 0

x,x>=0
```
```asciimath
abs(xy) = abs(x) * abs(y)

abs(x + y) <= abs(x) + abs(y)

abs(x - y) >= abs(x) - abs(y)
```
## Интересные примеры последовательностей

|`$(-1)^(n+1)$`|1, -1, 1, -1|
|`$(1+(-1)^n)/2$`|0, 1, 0, 1|
|`$((-1)^(n+1))/n^2$`|1, `$-1/4$`, `$1/9$`, `$-1/16$`|
