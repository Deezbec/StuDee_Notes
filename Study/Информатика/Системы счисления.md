### Определение коррекции
для x-y ичной 
Пример для 2-9
1) перевести (y - 1) в систему x
`$8_10 = 1000_2$`
2) Если `$(y-1)_x$` не состоит только из максимальных цифр СС, то все числа > `$(y-1)_x$` будут запрещенными
forbidden - `$1001_2$`(`$9_10$`) `$1010_2$`(`$10_10$`) `$1011_2$`(`$11_10$`) `$1100_2$`(`$12_10$`) `$1101_2$`(`$13_10$`) 
`$1110_2$`(`$14_10$`) `$1111_2$`(`$15_10$`)
### Правила коррекции
При сложении (+6 в 2-10)
+ попадание в запрещенную комбинацию
+ перелет через запрещенную область
  
При вычитании (-6 в 2-10)
+ заем из старшей тетрады
+ попадение в ЗО (запрещенную область)

Перед примером введем условные обозначения:
+ `$x^(**)$` - Попало в запрещенную область
+ `$x^(** **)$` - Перелетело запрещенную область
